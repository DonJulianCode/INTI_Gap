## üî¨ Descripci√≥n General

**Inti-Gap** es un pipeline de Materials Informatics dise√±ado para predecir la brecha energ√©tica (Band Gap) de compuestos inorg√°nicos mediante t√©cnicas de Machine Learning. El proyecto busca acelerar el proceso de descubrimiento de nuevos materiales para celdas solares, reduciendo significativamente la necesidad de costosas pruebas experimentales de laboratorio.

### Motivaci√≥n

El desarrollo de nuevos materiales fotovoltaicos es un proceso tradicionalmente lento y costoso. Inti-Gap permite realizar un filtrado r√°pido de candidatos viables, optimizando recursos y tiempo en la investigaci√≥n de materiales para energ√≠a solar.

## ‚ú® Caracter√≠sticas Principales

Inti-Gap implementa un flujo de trabajo completo que incluye:

1. **Ingesta de Datos**: Procesamiento de f√≥rmulas qu√≠micas en formato texto (ej: `GaAs`, `SnO2`, `CdTe`)

2. **Extracci√≥n Autom√°tica de Caracter√≠sticas**: Generaci√≥n de descriptores f√≠sico-qu√≠micos utilizando la biblioteca `matminer`:
   - Masa at√≥mica promedio

3. **Predicci√≥n de Band Gap**: Clasificaci√≥n autom√°tica del material como:
   - **Metal** (Band Gap ‚âà 0 eV)
   - **Semiconductor** (Band Gap: 0.5 - 4.0 eV)
   - **Aislante** (Band Gap > 4.0 eV)

### 1. Cargar el Modelo Pre-entrenado

```python

import pandas as pd
from matminer.featurizers.conversions import StrToComposition
from matminer.featurizers.composition import ElementProperty

print("‚úÖ Modelo Inti-Gap cargado exitosamente")
```

### 2. Funci√≥n de Predicci√≥n

```python
def predecir_band_gap(formula_quimica):
    """
    Predice el Band Gap de un material a partir de su f√≥rmula qu√≠mica.
    
    Args:
        formula_quimica (str): F√≥rmula qu√≠mica del compuesto (ej: 'GaAs', 'SnO2')
    
    Returns:
        float: Band Gap predicho en electronvoltios (eV)
    """
    # Crear DataFrame temporal
    df_temp = pd.DataFrame({"formula": [formula_quimica]})
    
    # Feature Engineering: Convertir texto a composici√≥n qu√≠mica
    df_temp = StrToComposition().featurize_dataframe(df_temp, "formula")
    
    # Extraer propiedades elementales (132 caracter√≠sticas)
    ep_feat = ElementProperty.from_preset(preset_name="magpie")
    df_temp = ep_feat.featurize_dataframe(df_temp, col_id="composition")
    
    # Preparar entrada para el modelo
    X_input = df_temp.drop(columns=['formula', 'composition'])
    
    # Realizar predicci√≥n
    prediccion = model.predict(X_input)[0]
    return prediccion
```

### 3. Ejemplo de Uso

```python

# Clasificaci√≥n autom√°tica

    print("üìä Clasificaci√≥n: Metal/Conductor")

    print("üìä Clasificaci√≥n: Semiconductor")

```
